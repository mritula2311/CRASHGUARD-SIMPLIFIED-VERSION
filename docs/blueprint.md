# **App Name**: CRASH GUARD

## Core Features:

- Location Display: Display crash location on a map using GPS data.
- Severity Rating: Display crash severity based on vibration sensor data.
- Speed Indicator: Display crash speed based on accelerometer data.
- Smart Alert Generation: Use a tool to Generate an emergency alert message with crash details and location; consider which recipients should receive each alert based on crash severity.
- Emergency Contact Customization: Enable users to customize emergency contact information
- Cloud Integration: Store sensor data (vibration, GPS, accelerometer) in Firebase Cloud.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to convey trust and security.
- Background color: Light Gray (#F0F2F5), nearly desaturated blue, for a clean, neutral backdrop.
- Accent color: Amber (#FFC107) to highlight critical information and alerts.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Use clear and universally recognized icons for location, severity, and speed.
- Prioritize critical information at the top, such as location and severity. Use a card-based layout for sensor data.
- Use subtle animations to draw attention to alerts, such as a pulsing effect on the severity indicator.
